# ATOP Maltego Transform
## REQUEREMENTS
To run ATOP Maltego transform you need:
- Python3 and pip 
- Install atop `pip install "atop>=0.0.2-08"`
- Install Maltego CE
- Clone this repo
- Install Entities from the packege atop.mtz
- Create a new local transform in Maltego CE
